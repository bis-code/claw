# Skill: wind-down

Daily brain dump and coding cutoff protocol to help prevent late-night coding sessions.

## Trigger
- `/wind-down` - Run end-of-day brain dump and log work

## Description
Helps capture daily progress and priorities before ending the workday:
1. Scans work and personal repos for today's commits
2. Prompts for quick brain dump (work done, personal work, tomorrow's priorities)
3. Captures energy level and focus stealers
4. Saves structured log to Obsidian vault
5. Reminds user to stop coding if after 10pm

Designed for ADHD-friendly workflow with time-boxed prompts and structured format.
