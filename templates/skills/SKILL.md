# Claw - Work Management

Simple work management with 4 commands.

## Commands

| Command | Purpose |
|---------|---------|
| `/run` | Start a work session |
| `/report-bug` | Report a bug |
| `/new-feature` | Propose a new feature |
| `/new-improvement` | Suggest an improvement |

## Usage

```
/run              - Select and work on items
/run 123          - Work on GitHub issue #123
/report-bug       - Create a bug report
/new-feature      - Propose a feature
/new-improvement  - Suggest a refactor/improvement
```
